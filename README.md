### STATS 211: Stochastic Process Course Project
- Topic: Gold VaR (Value at Risk) and ES (Expected Shortfall) Analysis Based on Bayesian Risk Analysis
- Data source: https://www.investing.com/commodities/gold-historical-data, from Apr. 2, 2022 to Apr. 2, 2026
- Research Method: t distribution, Metropolis within Gibbs
- Tools: Python (numpy, pandas, matplotlib, seaborn, yfinance), AI tools (Gemini, Kimi K2, Claude Code, etc.)
- Expected timeline:
  1. Fetching data set and pre-processing (w3);
  2. Derive the Full Conditional Posterior Distributions for the t-distribution parameters (w4);
  3. Core algorithm (Develop the Metropolis-within-Gibbs sampler) (w5-w6);
  4. Calculate VaR and ES, uncertainty quantification, find credible intervals (w5-w6);
  5. Report, presentation, submission (w7-w8);
